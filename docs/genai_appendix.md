# GenAI Appendix

## How Generative AI Was Used
Generative AI was used in a limited support role during this project. Its main use was to help review implementation choices, check code structure, tighten documentation wording, and suggest debugging directions when individual components did not behave as expected.

## What It Was Not Used For
Generative AI was not used as a substitute for building the project from scratch. The system design, implementation decisions, data flow, model setup, and final verification were completed manually and checked against the assignment requirements.

## Verification Approach
- I reviewed and edited any AI-assisted suggestions before using them.
- I validated code paths locally through scripts, artifacts, and consistency checks.
- I kept the final project structure aligned with the assignment brief and my own reporting conventions.
