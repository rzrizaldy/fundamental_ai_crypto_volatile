Methods / Log

Short implementation notes in the same spirit as the reference dashboard.

Data source: Coinbase Advanced Trade public market-data WebSocket.

Transport: Kafka raw and feature topics with local Docker Compose infrastructure.

Feature cadence: 1-second windows with a 60-second future-volatility label horizon.

Modeling: z-score baseline and logistic regression with MLflow tracking.
